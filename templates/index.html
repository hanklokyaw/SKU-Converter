SKU Converter

Note: In the new SKU, we use the letter "l" instead of "/". For example, "3/4" becomes "3l4".

Example:
1. Use "Fanfare3?" to search any information that starts with Fanfare3.
2. Use "?1.25" to search any information that ends with 1.25.
3. Use "?TOP?" to search any information that includes TOP.
